# Exemples de pratique professionnelle

> Page titre obligatoire, suivie des exemples illustrant ta pratique pour chaque compétence à attester.

Les exemples de pratique professionnelle ci-après s'appuient principalement sur le projet **Périnéa** (application mobile de rééducation périnéale connectée), ainsi que sur `[À COMPLÉTER : autres projets / expériences en entreprise]`.

Pour chaque compétence (voir fiches suivantes), l'exemple décrit :
- le **contexte** (projet, environnement, contraintes) ;
- la **tâche réalisée** ;
- les **moyens et technologies** mis en œuvre ;
- le **résultat** obtenu.
